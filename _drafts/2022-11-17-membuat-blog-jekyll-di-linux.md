## Apa Itu Jekyll

```
├── _includes
├── _layouts
├── _pages
├── _post
├── _site
``` 
    
## Memasang Jekyll di Linux

```
$ sudo apt -y install ruby ruby-dev
$ gem install bundler
$ gem install jekyll
```

## Membuat Blog Jekyll

```
$ jekyll serve
```
